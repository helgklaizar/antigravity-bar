# 🌍 Global Professions & Skills Taxonomy

*Based on an analysis of 2,500+ skills and 150+ agents across community repositories (`VoltAgent`, `sickn33`, etc.).*

The `AI-Ecosystem` is expanding from a personal toolkit into a **Global AI Library for Everyone**. Based on the massive dataset of external skills we mapped, here is the official categorization and analysis of all professions our ecosystem will support. 

When users request skills, they will be pulled from these exact categorical folders within our repository.

---

## 1. 🌐 Web & Mobile Frontend
**Target Audience:** UI/UX Developers, Mobile App Engineers.
**Available Skills/Agents:** React Performance, Tailwind Systems, Flutter Experts, SwiftUI Refactoring, Expo CI/CD, Next.js Templates, ThreeJS Materials, WebGL.
**Ecosystem Path:** `skills/frontend/`

## 🛠️ Execution Strategy (Next Steps)

1. We will create these 10 core category folders in our `skills/` and `agents/` directories.
2. We will programmatically migrate and standardize all 2,500+ skills from the community repos into this folder structure.
3. **Important:** These skills will sit in the `AI-Ecosystem` repository as a library. They will *not* be automatically injected into local user projects unless explicitly requested via their specific profession profile in `AI_ONBOARDING.md`.
